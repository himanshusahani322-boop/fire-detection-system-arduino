# 🔥 Fire Detection System using Arduino

This project is a **Smoke & Fire Detection System** built using Arduino that detects smoke levels and provides **multi-level alerts** for safety.

##  Features

* Real-time smoke detection
* 4-Level alert system:

  *  Safe Zone (0–250)
  *  Detected Zone (251–400)
  *  Alert Zone (401–500)
  *  Danger Zone (501–800)
* Fast response system

##  Components Used

* Arduino Uno
* Smoke Sensor (MQ-2)
* Breadboard
* Jumper Wires

##  Working Principle

The smoke sensor continuously reads air quality. Based on the sensor values:

* Low values → Safe
* Medium values → Warning
* High values → Fire Alert
